# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| 0.1.x (latest release) | ✅ |
| `main` | ✅ |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

- Email: **mail@030.dev** (subject: `[SECURITY] vaultrun`)
- Or use GitHub's [private vulnerability reporting](https://github.com/nickvd7/vaultrun/security/advisories/new)

Include a description of the issue, steps to reproduce, and the affected component
(API server, MCP server, CLI, CI runner, dashboard, or SDK). You can expect an
initial response within a few days.

## Security model

VaultRun's threat model, container hardening, and production checklist are documented
in [docs/security.md](docs/security.md).
